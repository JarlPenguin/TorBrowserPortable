Tor Browser Portable Help

Anonymous browsing, wherever you go.

Tor Browser Portable is a portablized version of the Tor Browser, which allows for truly anonymous browsing. It's packaged with a PortableApps.com launcher as a portable app, so you can use it from a cloud folder, portable drive, or local folder without needing to install it on each PC.
